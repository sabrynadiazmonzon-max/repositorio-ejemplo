# Repositorio-ejemplo

Este es un repositorio simple para practicar Git y GitHub.  
Aquí se muestran ejemplos básicos de archivos y carpetas.

## Contenido
- `hola.txt`: archivo de prueba.
- Carpeta `imagenes/`: carpeta vacía para mostrar estructura.

## Objetivo
Aprender lo básico:
- ¿Qué es un repositorio?
- ¿Cómo subir cambios?
- ¿Cómo crear commits?
- ¿Cómo organizar archivos?
